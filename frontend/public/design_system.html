Typography

We use a modern font stack. For PPTs, you may substitute with standard system fonts if necessary.

Primary Font (Headings & Body)
Geist Sans / Inter

Used for all primary headings and body text. Clean, modern, and highly legible. If unavailable, use Arial or Helvetica.

Monospace Font (Code & Data)
Geist Mono / JetBrains Mono

Used for code snippets, data tables, and technical readouts. If unavailable, use Courier New.
